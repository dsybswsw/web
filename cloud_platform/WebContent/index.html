Brief Introduction

This platform implements some data mining algorithm and users can upload their own data to it and get the computation result.

Algorithms

Classification

Classification is a set of supervised learning algorithms. Given some labeled training data, a classifier will train a classification model and the model will be used to predict future data.

Clustering

Clustering is set of unsupervised learning algorithms. Given some unlabeled data, a clustering algorithm will divided data instance to different clusters in hard or soft ways.
